# Seeds

Use this skill when the task involves creating, updating, browsing, or resolving issues tracked by Seeds.

Preferred commands:

```bash
seeds --help
seeds list
seeds status
```

Rules:
- Use seeds for durable issues and task records.
- Keep issue updates factual and minimal.
- Prefer updating the tracked issue rather than repeating status in chat.
